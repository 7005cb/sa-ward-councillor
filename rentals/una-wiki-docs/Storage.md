It often happens when you need add an image or a link to a file to some site's page. In this case, the Storage app will help you.

In the app there are two lists with similar functionality: 
* Files 
* Images

In each of these lists you can upload an unlimited number of files (by clicking the Add button in the right upper part of the page), which can later be used on the pages of the site.

After uploading  file, the following operations are available:
* Removal
* Download
* Resize (for images only)

In the URL column there is field with link to a resource (file or image) that can be used in page's content creation.
