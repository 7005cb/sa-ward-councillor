For manage UNA accounts need install App Accounts Manager. ([Learn more about installing apps](https://github.com/unaio/una/wiki/Add-Apps))

![](https://raw.githubusercontent.com/wiki/unaio/una/images/app-accounts-manager/001.png)

On the Accounts Manager settings page you can see a list of all accounts that were registered on the system.
You can Activate / Suspend account by using switcher.

![](https://raw.githubusercontent.com/wiki/unaio/una/images/app-accounts-manager/002.png)

To show action menu for selected account  need to click the icon in the far right column:

![](https://raw.githubusercontent.com/wiki/unaio/una/images/app-accounts-manager/004.PNG)

Available actions:

* Edit Email - allows to change account email address, as well as change the status of receiving information about system updates and newsletters.
* Resend Confirmation - resend the Confirmation letter to account email address.
* Reset Password - allows change the  account's password 
* Delete - remove account from UNA
* Delete With Content - remove account and all related content from UNA 
* Make / Unmake Operator - gives operator status to account. Operator status allows [manage the UNA from Studio](https://github.com/unaio/una/wiki/Studio).

The following bulk actions are also available with accounts.

To select several accounts, need to check the checkboxes to the left of them

![](https://raw.githubusercontent.com/wiki/unaio/una/images/app-accounts-manager/003.png)

* Activate (for suspended accounts)
* Suspend (for activer accounts)
* Resend Confirmation - as described above
* Delete - as described above
* Delete With Content - as described above
