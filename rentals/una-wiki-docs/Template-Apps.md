Template Apps provide possibilities to have exclusive look of the site without changes outside of their folders. Every Template App offers to customize images / icons, CSS and HTML templates of pages / part of pages extended from parent ("base") template by own way for the whole site / only for limited places. UNA's team developed 3 templates: **Ocean**, **Protean** and **Decorous**. Every from this set is made to work flawlessy on any screen and any devices.

## Ocean

Design template for early development builds of UNA. Is fine for training purposes to learn all UNA's template principles.

## Protean

Highly-customisable template. Mobile-friendly horizontal navigation. Wigh minimalist look, designed for various style combinations.

## Decorous

This one features a "conventional" persistent vertical navigation, with a responsive transition to icons-only and then "sandwich" menu with a pop-down list.
